Plugin Ecosystem

Extend with custom logic

Extend the Net Observation Project by registering plugins via window.registerPlugin. Plugins may attach new terminal commands, respond to lifecycle hooks, or enrich telemetry with overlays. This lightweight system encourages experimentation and allows operations teams to tailor workflows.

registerPlugin({
  name: 'intel-broadcast',
  command: 'broadcast',
  run(message, { log }) {
    log(`Broadcasting signal → ${message}`);
    return 'Signal routed to relay mesh.';
  }
});
